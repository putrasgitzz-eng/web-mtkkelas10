Eksponen

Pengertian Eksponen

Eksponen adalah bentuk perkalian berulang dari suatu bilangan yang sama. Bentuk umum eksponen adalah:

an = a × a × a × ... × a (sebanyak n kali)

Di mana:

  • a = basis (bilangan pokok)
  • n = eksponen (pangkat)

Sifat-Sifat Eksponen

1. am × an = am+n
2. am ÷ an = am-n
3. (am)n = am×n
4. (a×b)n = an × bn
5. (a/b)n = an / bn
6. a0 = 1 (dengan a ≠ 0)
7. a-n = 1/an

Contoh Soal

Soal: Sederhanakan 23 × 25

Penyelesaian:

Menggunakan sifat eksponen: am × an = am+n

23 × 25 = 23+5 = 28 = 256

Sistem Persamaan Linear Dua Variabel (SPLDV)

Pengertian SPLDV

SPLDV adalah sistem yang terdiri dari dua persamaan linear dengan dua variabel. Bentuk umum SPLDV:

a₁x + b₁y = c₁
a₂x + b₂y = c₂

Metode Penyelesaian SPLDV

  1. Metode Grafik: Menggambar garis dari setiap persamaan dan mencari titik potongnya.
  2. Metode Substitusi: Menyatakan satu variabel dalam variabel lain, lalu mensubstitusikan.
  3. Metode Eliminasi: Mengeliminasi salah satu variabel dengan menjumlahkan atau mengurangkan persamaan.
  4. Metode Gabungan: Menggabungkan metode substitusi dan eliminasi.

Contoh Soal

Soal: Tentukan nilai x dan y dari sistem persamaan:

2x + y = 7

x - y = 2

Penyelesaian (Metode Eliminasi):

Tambahkan kedua persamaan:

(2x + y) + (x - y) = 7 + 2

3x = 9 → x = 3

Substitusi x = 3 ke persamaan kedua:

3 - y = 2 → y = 1

Jadi, solusinya adalah x = 3 dan y = 1

Persamaan dan Fungsi Kuadrat

Bentuk Umum

Persamaan kuadrat memiliki bentuk umum:

ax² + bx + c = 0, dengan a ≠ 0

Fungsi kuadrat memiliki bentuk umum:

f(x) = ax² + bx + c, dengan a ≠ 0

Rumus ABC

Untuk menyelesaikan persamaan kuadrat ax² + bx + c = 0, dapat digunakan rumus ABC:

x = [-b ± √(b² - 4ac)] / (2a)

Diskriminan (D) = b² - 4ac menentukan jenis akar:

  • D > 0: Dua akar real berbeda
  • D = 0: Dua akar real sama (kembar)
  • D < 0: Tidak memiliki akar real

Grafik Fungsi Kuadrat

Grafik fungsi kuadrat berbentuk parabola dengan sifat:

  • Jika a > 0, parabola terbuka ke atas
  • Jika a < 0, parabola terbuka ke bawah
  • Titik puncak: (-b/2a, -D/4a)
  • Sumbu simetri: x = -b/2a

Persamaan dan Fungsi Eksponensial

Pengertian

Fungsi eksponensial adalah fungsi yang variabel bebasnya merupakan eksponen. Bentuk umum:

f(x) = a × bx

Di mana:

  • a = koefisien
  • b = basis (b > 0, b ≠ 1)
  • x = eksponen

Sifat-Sifat Fungsi Eksponensial

  • Grafik selalu melalui titik (0, a)
  • Jika b > 1, fungsi naik (monoton naik)
  • Jika 0 < b < 1, fungsi turun (monoton turun)
  • Sumbu x sebagai asimtot horizontal

Persamaan Eksponensial

Persamaan eksponensial adalah persamaan yang eksponennya mengandung variabel. Bentuk umum:

af(x) = ag(x) → f(x) = g(x)

Contoh: Selesaikan 2x+1 = 8

Penyelesaian:

8 = 2³, sehingga:

2x+1 = 2³

x + 1 = 3

x = 2

Trigonometri

Perbandingan Trigonometri

Dalam segitiga siku-siku:

sin θ = sisi depan / sisi miring
cos θ = sisi samping / sisi miring
tan θ = sisi depan / sisi samping

Perbandingan lainnya:

cosec θ = 1/sin θ
sec θ = 1/cos θ
cot θ = 1/tan θ

Identitas Trigonometri

sin² θ + cos² θ = 1
1 + tan² θ = sec² θ
1 + cot² θ = cosec² θ
sin(2θ) = 2 sin θ cos θ
cos(2θ) = cos² θ - sin² θ

Statistika

Box Plot (Diagram Kotak)

Box plot adalah diagram yang menampilkan ringkasan distribusi data berdasarkan:

  • Nilai minimum
  • Kuartil bawah (Q1)
  • Median (Q2)
  • Kuartil atas (Q3)
  • Nilai maksimum

Diagram Pencar (Scatter Plot)

Diagram pencar digunakan untuk melihat hubungan antara dua variabel. Pola yang mungkin:

  • Hubungan positif: Ketika x meningkat, y juga meningkat
  • Hubungan negatif: Ketika x meningkat, y menurun
  • Tidak ada hubungan: Titik-titik tersebar acak

Contoh Interpretasi

Box Plot: Jika median berada di tengah kotak dan panjang whisker seimbang, distribusi data simetris.

Scatter Plot: Jika titik-titik membentuk pola menaik dari kiri ke kanan, terdapat korelasi positif antara variabel.
